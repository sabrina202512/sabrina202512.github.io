Project Description

This project is a mini hotel booking website created as a final assessment for the Web Design course. The website demonstrates knowledge of HTML5 and CSS3 through multiple pages, responsive layout, image galleries, forms, tables, and semantic accessibility.

The goal was to build a functional, well-organized, and visually appealing website that allows users to navigate, explore rooms, and submit a booking request easily.

What I Learned

  • How to build multi-page websites with internal linking
  • Using Flexbox for layout and spacing
  • Creating accessible and valid forms
  • Applying semantic HTML and clean CSS styles
  • Hosting with GitHub Pages and organizing files
